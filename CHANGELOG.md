# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-XX

### Added
- Initial release of Universal Leaderboard
- Multi-game leaderboard support
- Win count and total points scoring modes
- PWA support with offline functionality
- Statistics and history tracking
- Built-in Snake game with virtual controls
- Built-in Memory game with multiple difficulty levels
- Responsive design for mobile and desktop
- Retro-themed UI design
- Data export/import functionality
- Sound effects
- Bottom navigation bar
- Settings page with sound toggle and data management

### Features
- Create multiple leaderboards for different games
- Add/remove players from leaderboards
- Update scores with custom values
- View leaderboard statistics and rankings
- Track leaderboard history and activity
- Play built-in retro games (Snake, Memory)
- Install as PWA on mobile devices
- Export/import data as JSON

### Technical
- React 18.3.1
- TypeScript 5.8.3
- Vite 7.2.2
- React Router 6.30.1
- shadcn-ui components
- Tailwind CSS for styling
- PWA support with Workbox
- Service worker for offline functionality
